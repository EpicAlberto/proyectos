# Política de Privacidad de SalvApp

**Última actualización:** 28 de junio de 2026

Edy Alberto Hernández Martínez ("el Desarrollador") ha creado la aplicación **SalvApp** como una aplicación gratuita y de carácter educativo. Este servicio se proporciona sin costo alguno y está diseñado para ser utilizado tal cual.

Esta página se utiliza para informar a los visitantes sobre las políticas con respecto a la recopilación, el uso y la divulgación de Información Personal de quienes deciden utilizar este servicio.

## 1. Recopilación y Uso de la Información
**SalvApp NO recopila, almacena, ni comparte información personal o sensible del usuario.** La aplicación no requiere que el usuario cree una cuenta, inicie sesión o proporcione datos identificables (como nombre, correo electrónico, ubicación, etc.) para utilizar sus funciones principales.

## 2. Permisos de la Aplicación
La aplicación únicamente puede requerir acceso a permisos básicos del sistema (como el acceso a Internet) exclusivamente para cargar o consultar la información de bases de datos externas de manera temporal.

## 3. Uso de Información Externa y Descargo de Responsabilidad Médica
SalvApp utiliza información pública y de libre acceso proveniente de **MedlinePlus** con fines puramente educativos e informativos. 
* **No es un dispositivo médico.**
* **No sustituye el diagnóstico, asesoramiento o tratamiento de un profesional de la salud.**
Cualquier decisión tomada basándose en la información de esta aplicación es responsabilidad exclusiva del usuario.

## 4. Enlaces a Otros Sitios
Este Servicio puede contener enlaces a otros sitios (como repositorios médicos). Si hace clic en un enlace de un tercero, será dirigido a ese sitio. Tenga en cuenta que estos sitios externos no son operados por el Desarrollador. Por lo tanto, se recomienda encarecidamente revisar la Política de Privacidad de estos sitios web. 

## 5. Cambios en esta Política de Privacidad
El Desarrollador puede actualizar esta Política de Privacidad de vez en cuando. Por lo tanto, se le recomienda revisar esta página periódicamente para ver si hay cambios. Se le notificará de cualquier cambio publicando la nueva Política de Privacidad en esta página.

## 6. Contacto
Si tiene alguna pregunta o sugerencia sobre esta Política de Privacidad, no dude en contactarme a través de los canales oficiales proporcionados en la página de la aplicación en Google Play Store.
